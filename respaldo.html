Respaldos

Descarga, restaura y monitorea el espacio en Supabase

📊 Espacio utilizado

El plan gratuito de Supabase incluye 500 MB de base de datos.

⬇️ Descargar respaldo

Genera un JSON con toda la información: sucursales, planes, clientes, membresías, pagos, asistencias e inventario.

♻️ Restaurar respaldo

Sube un JSON generado por esta misma herramienta. Los registros con el mismo ID se actualizan; los nuevos se crean. No elimina datos actuales.
