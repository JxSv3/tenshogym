Respaldos y almacenamiento

Descarga, restaura tu información y revisa el espacio usado en Supabase

📊 Espacio utilizado en la base de datos

El plan gratuito de Supabase incluye 500 MB de base de datos. Ajusta este valor si tu plan es distinto.

⬇️ Descargar respaldo

Genera un archivo JSON con toda la información: sucursales, planes, clientes, membresías, pagos y asistencias.

♻️ Restaurar respaldo

Sube un archivo JSON generado por esta misma herramienta. Los registros existentes con el mismo ID serán actualizados; los que no existan se crearán. No elimina datos actuales que no estén en el archivo.
